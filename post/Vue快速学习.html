title:Vue快速学习

1.1初识Vue

Vue简介

✔ Vue是一套用来动态构建用户界面的渐进式Javascript框架

​ 构建用户界面:把数据通过某种办法变成用户界面

​ 渐进式：自底向上的逐层应用，简单应用只需要一个轻量小巧的核心库，复杂应用可以引入各式各样的Vue插件

✔Vue的特点：遵循MVVM模式；

​ 编码简洁体积小，效率高，本身只关注UI，可以引入第三方库开发项目；

​ 采用组件化模式，提高代码复用率（易于维护）；

​ 声明式编码，无需直接操作DOM

​ 使用DOM和DIFF算法，提高DOM节点的复用率

​ （DOM：是HTML和XML文档的编程接口，是一种处理HTML和XML文件的标准API。

✔Vue初始操作：

  • 通过创建一个Vue实例，并往该实例中传入一个配置对象，才能让Vue工作
  • 容器里面的代码要符合html规范，里面的代码被称为Vue模板
  • Vue实例和容器是一一对应的
  • 里面的xxx应该是js表达式（产生值），且xxx可以自动读取到data中的所有属性
  • data中的数据发生变化，模板中的数据也会发生变化。

1.5 MVVM模型和数据代理

MVVM模型：

  • M：model模型，data中的数据
  • V：view视图，模板代码
  • VM：ViewModel视图模型，Vue实例

​

数据代理

通过一个对象代理对另一个对象中属性的操作

原理：

  • 通过object.defineProperty()把data对象中所有属性添加到vm上
  • 为每一个添加到vm上的属性，都指定一个getter和setter
  • 在getter和setter内部去操作data中对应的属性

1.6 事件处理

基本语法

  1. 用v-on:xxx绑定事件（简写 @xxx)
  2. 事件的回调需要配置在methods对象中，最终会在vm上
  3. methos中配置的函数，不能用箭头函数，不然this就不是vm（会是指针的指针？）
  4. this指向的都是vm或者组件实例对象
  5. @click=”demo” 和 @click=”demo($event)”效果一直，但后者可以传参
<!DOCTYPE html>
<html lang = "en">
    <head>
        <meta charset = "UTF-8">
        <meta name = "viewport" content = "width=device-width,initial-scale=1.0">

事件修饰符

  • prevent：阻止默认事件

  • stop：阻止事件冒泡(事件冒泡是指一个事件从发生的元素开始，沿着 DOM 树向上传播，直到最外层的父元素。这个过程中，所有的祖先元素都可以监听和响应这个事件。1234

    例如，如果你点击了一个按钮，这个按钮的点击事件会先触发，然后会依次冒泡到它的父元素，祖父元素，直到 document 和 window 对象。2)

  • once：让事件只发生一次

  • self：中有event.target是当前操作的元素的时候才会触发

  • 修饰符可以连续使用

    @click.prevent.stop="showinfo"

      3. 可以用keyCode

      4. 去指定jurisdiction的按键
